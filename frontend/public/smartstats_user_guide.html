Data Input
02

Uploading Data

  • Upload a CSV file directly from your device
  • Or paste CSV content directly into the input box
  • First row must contain column headers
  • Values must be comma-separated
  • Maximum file size: 5 MB
  • If format is incorrect, a validation error will describe the issue

Best Practices
06

Tips & Best Practices

📌 Data Format

  • Use clear, descriptive column names
  • Avoid special characters in headers
  • Keep data types consistent — numbers vs text
  • Ensure the first row contains only headers

🗂️ Organization

  • Create one dashboard per project
  • Use descriptive names for charts
  • Delete unused charts and dashboards
  • Rearrange charts for cleaner layouts
Troubleshooting
07

Troubleshooting

Problem
File won't upload
Solution
Check file type, size (max 5MB), and that the CSV is properly formatted
Problem
Charts show "Item 1, Item 2"
Solution
Re-upload your data — the first row must contain proper column headers
Problem
Login session expired
Solution
Sessions last 24 hours. Simply log in again to continue working
Problem
Analysis is slow
Solution
Large files can take 10–30 seconds to process. This is expected behavior
